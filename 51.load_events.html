Load Events

onload (display alert)

Display an alert when the page has been loaded:


			//onload event attached to body element
			function myFunction1() {
				alert ("The page has been loaded!");
			}

onload (display alert)

Display an alert when an image has been loaded:


			//onload event attached to image element
			function myFunction2() {
				alert("The image has been loaded!");
			}

onload (display error)

Display an error message (alert) when there is an error loading an image:


			//onload event attached to image element
			function imgError() {
				alert("The image could not be loaded!");
			}

onunload (display a message)

Display an error (alert) when the window is closed (or press F5 to reload):


			//onunload event attached to body element
			function myFunction3() {
				alert("Thanks for visiting!");
			}

onresize (display dimensions)

Display the dimensions of the window when it is resized: (resize the window to see the effect!)


			//onresize event attached to body element
			function myFunction4() {
				var w = window.outerWidth;
				var h = window.outerHeight;
				var txt = "Window size: width=" + w + " , height=" + h;

				document.getElementById("demo").innerHTML = txt;
			}
