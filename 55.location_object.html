Location Object

The window.location object can be used to get the current page address (URL) and to redirect the browser to a new page. It can be written without the window prefix.

Hostname/Port

Return the hostname and port of the current URL:


			document.getElementById("demo1").innerHTML =
			"The hostname is: " + window.location.hostname;

URL

Return the entire URL of the current page:


			document.getElementById("demo2").innerHTML =
			"The page hostname is: " + window.location.href;

Pathname

Return the pathname of the current URL:


			document.getElementById("demo3").innerHTML =
			"The pathname is: " + window.location.pathname;

Protocol

Return the protocol portion of the current URL:


			document.getElementById("demo4").innerHTML =
			"The page protocol is " + window.location.protocol;

Load Document

Load a new document:

Note: this will open a new document in this window...click the browser's "back" button to return to this page!


			function newDoc() {
				window.location.assign("http://www.bing.com");
			}

Replace Document

Replace the current document:

Note: this will replace this document. Come back to this page to see more examples!


			function replaceDoc() {
				window.location.replace("http://www.google.com");
			}

Break Frame

Break out of a frame:

Note: this will break out of the current frame...use the browser's back button to return to this page!


			function breakOut() {
				if (window.top != window.self) {
					window.top.location = "demo_iframe2.html";
				}
			}
