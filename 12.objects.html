Objects

Objects are variables, but objects can contain many values. The values are written as name:value pairs (separated by semicolon). JavaScript objects are containers for named values.

Create a JavaScript Variable


			var car = "Audi";
			document.getElementById("demo1").innerHTML = car;

Create a JavaScript Object


			var car = {type: "fiat", model: "500", color: "white"};
			document.getElementById("demo2").innerHTML = car.color;

Create a Person Object (single line)


			var person = {firstName:"John", lastName:"Doe", age: 49, eyeColor:"blue"};

			document.getElementById("demo3").innerHTML =
			person.firstName + " is " + person.age + " years old.";

Create a Person Object (multiple lines)


			var person = {
				firstName: "John",
				lastName: "Doe",
				age: 55,
				eyeColor: "blue"
			};

			document.getElementById("demo4").innerHTML =
			person.firstName + " is " + person.age + " years old.";

Accessing Object Properties Using .property


			var person = {
				firstName: "John",
				lastName: "Doe",
				age: 45,
				eyeColor: "blue"
			};

			document.getElementById("demo5").innerHTML = person.lastName;

Accessing Object Properties Using [property]


			var car = {
				carName: "Audi",
				carColor: "Blue",
				carStyle: "Sport"
			};

			document.getElementById("demo6").innerHTML =
			car["carStyle"];

Accessing a Function Property as a Method


			var person = {
				firstName: "Joe",
				lastName: "Atwood",
				age: 37,
				eyeColor: "blue",
				fullName: function() {
					return this.firstName + " " + this.lastName;
				}
			};

			document.getElementById("demo7").innerHTML = person.fullName();

Access a Function Property as a Property


			var person = {
