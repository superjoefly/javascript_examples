JavaScript Booleans

A JavaScript boolean represents either true or false.

Value of Boolean

Display the value of boolean (10 > 9):


			function myFunction1() {
				document.getElementById("demo1").innerHTML = Boolean(10 > 9);
			}

Value of Boolean

Display the value of boolean (10 < 9):


			function myFunction2() {
				document.getElementById("demo2").innerHTML =
				Boolean(10 < 9);
			}

Real Values

Everything with a real value (except 0) is true:


			document.getElementById("demo3").innerHTML =
			Boolean(100) +
			Boolean(3.14) +
			Boolean(-20) +
			Boolean("World") +
			Boolean('false') +
			Boolean(1 + 8 + 3.14) +
			Boolean("0") +
			Boolean(0) +
			Boolean(-0);

			//add line breaks!

Empty String

The boolean value of an empty string is false:


			document.getElementById("demo4").innerHTML =
			Boolean("");

Undefined

The boolean value of undefined is false:


			var x;
			document.getElementById("demo5").innerHTML =
			Boolean(x);

Null

The boolean value of null is false:


			var x = null;
			document.getElementById("demo6").innerHTML =
			Boolean(x);

False

The boolean value of false is false:


			var x = false;
			document.getElementById("demo7").innerHTML =
			Boolean(x);

NaN

The boolean value of NaN is false:


			var x = 10 / "j";
			document.getElementById("demo8").innerHTML =
			Boolean(x);
