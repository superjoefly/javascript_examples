Navigation Object

The window.navigator object contains information about the visitor's browser. It can be written without the window prefix.

Cookies

Check if cookies are enabled:


			document.getElementById("demo1").innerHTML =
			navigator.cookieEnabled;

Browser Name

Get the name of the visitor's browser:

"Mozilla" is the application code name for Chrome, Firefox, IE, Safari, and Opera.


			document.getElementById("demo2").innerHTML = navigator.appCodeName;

Product Name

Get the product name of the visitor's browser:


			document.getElementById("demo3").innerHTML = navigator.product;

Version

Get the version information of the visitor's browser:


			document.getElementById("demo4").innerHTML = navigator.appVersion;

User Agent

Get the user agent information for a visitor's browser:


			document.getElementById("demo5").innerHTML = navigator.userAgent;

Platform

Get the platform of the visitor's operating system:


			document.getElementById("demo6").innerHTML = navigator.platform;

Language

Get the language of the visitor's browser:


			document.getElementById("demo7").innerHTML = navigator.language;

Java

Check to see if Java is enabled:


			document.getElementById("demo8").innerHTML = navigator.javaEnabled();
