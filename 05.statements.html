JavaScript Statements

In HTML, JavaScript statements are "instructions" to be executed by the browser:

Commands


			document.getElementById("demo1").innerHTML = "Well, hello there!";

Sequence

JavaScript code is a sequence of JavaScript statements:


			var x, y, z;
			x = 5;
			y = 8;
			z = x + y;
			document.getElementById("demo2").innerHTML = z;

Separation

JavaScript statements are separated with a semicolon:


			var a, b, c;
			a = 5;
			b = 4;
			c = a * b;
			document.getElementById("demo3").innerHTML = c;

Multiple Statements

Mutiple statements on one line is allowed:


			var a, b, c;
			a = 5; b = 2; c = 3 * (a + b);
			document.getElementById("demo4").innerHTML = c;

Code Blocks

JavaScript statements can be grouped together in code blocks. Code blocks are written between mustaches {} (curly brackets):


			function myFunction() {
				document.getElementById("demo5").innerHTML = "Well, hello there!";

				document.getElementById("demo6").innerHTML = "How are you?";
			}

Breaking Code Lines

It is best to break a long line of code after an operator or a comma:


			document.getElementById("demo7").innerHTML =
			"This is a long line of code because this is sentence is fairly long, so I broke the line of code at the comma!";
