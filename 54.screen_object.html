Screen Object

The window.screen object contains information about the user's screen. It can be written without the window prefix...

Screen Width

Display the width of the screen:


			document.getElementById("demo1").innerHTML =
			"The screen width is: " + screen.width;

Screen Height

Display the height of the screen:


			document.getElementById("demo2").innerHTML =
			"The screen height is: " + screen.height;

Available Width

Display the available width of the screen:


			document.getElementById("demo3").innerHTML =
			"The available width of the screen is: " + screen.availWidth;

Available Height

Display the available height of the screen:


			document.getElementById("demo4").innerHTML =
			"The available height of the screen is: " + screen.availHeight;

Color Depth

Display the color depth of the screen:


			document.getElementById("demo5").innerHTML =
			"The color depth of the screen is: " + screen.colorDepth;

Pixes Depth

Display the pixel depth of the screen:


			document.getElementById("demo6").innerHTML =
			"The pixel depth of the screen is: " + screen.pixelDepth;
