JavaScript Comments

JavaScript comments can be used to explain JavaScript code, and to make it more readable.

JavaScript comments can also be used to prevent execution, for example, when testing alternative code.

Single Line Comments

To comment a single line use // comments go here:

Note: the comments are not executed and will not be shown in the broser window.


			//document.getElementById("demo1").innerHTML = "not executed!";
			document.getElementById("demo1").innerHTML = "executed!";

Single Line Comments at End of Line

Comments at the end of a line are often used to explain code:


			var x, y;
			x = 5;
			y = x * x;
			document.getElementById("demo2").innerHTML = y; //write y to demo2

Multi Line Comments

To comment multiple line (long lines etc) use /* comments go here */:


			/*The follwing code will change the paragraph with id="demo3" to include text saying "What going on today?"*/

			document.getElementById("demo3").innerHTML = "What's going on today?";

Prevent Execution

Comments can be used to prevent code execution (comment out).

The following code will not be executed because it is commented out:


			//document.getElementById("demo4").innerHTML = "This line of code will not be executed!";
