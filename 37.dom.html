Document Object Model

When a webpage is loaded, the browser creates a Document Object Model of the page.

The HTML DOM model is constructed as a tree of Objects:

The HTML DOM Tree of Objects

With the object model, JavaScript gets the power it needs to create dynamic HTML:

  • Change HTML elements
  • Change HTML attributes
  • Change CSS styles
  • Remove existing HTML elements and attributes
  • Add new HTML elements and attributes
  • React to HTML events
  • Create new events

HTML DOM

the HTML DOM is a standard object model for programming interface for HTML. It defines:

  • HTML elements as objects
  • Properties of HTML elements
  • Methods to access HTML elements
  • Events for HTML elements

The HTML DOM is a standard for how to get, change, add, or delete HTML elements.
