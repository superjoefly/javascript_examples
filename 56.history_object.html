History Object

The window.history object contains the browser's history. It can be written without the window prefix.

Number of URLs

Display the number of URLs in the history list:


			function urlNumb() {
				document.getElementById("demo1").innerHTML = history.length;
			}

Back Button

Create a back button on a page:


			function goBack() {
				window.history.back();
			}

Forward Button

Create a forward button on a page:


			function goForward() {
				window.history.forward();
			}

Go To History

Go to a specific URL from the history list:


			function goTo() {
				window.history.go(-2);
			}
