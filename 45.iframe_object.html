iFrame Object

The following iframe is used for the examples on this page:

Background Color:

Change the background color of the document in the iframe:

Note: background color of iframe (body) must be initially set to transparent on the demo page to work.


			function myFunction1() {
			    var x = document.getElementById("myframe");
			    x.style.backgroundColor = "blue";
			}

Height

Find the height of an iframe:


			function myFunction2() {
				var x = document.getElementById("myframe").height;
				document.getElementById("demo2").innerHTML = x;
			}

Name

Find the value of the name attribute of an iframe:


			function myFunction3() {
				var x = document.getElementById("myframe").name;
				document.getElementById("demo3").innerHTML = x;
			}

Get Source

Find the source attribute of an iframe:


			function myFunction4() {
				var x = document.getElementById("myframe").src;
				document.getElementById("demo4").innerHTML = x;
			}

Change Source

Change the source attribute of an iframe:


			function myFunction5() {
				document.getElementById("myframe").src="http://www.bing.com";
			}
