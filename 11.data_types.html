Data Types

JavaScript variables can hold many data types: numbers, string, objects and more:

Declare (create) Strings


			var carName1 = "Volvo";
			var carName2 = 'Audi';
			document.getElementById("demo1").innerHTML =
			carName1 + carName2; //add line breaks

Declare (create) Numbers


			var x1 = 34;
			var x2 = 34.00;
			var y = 123e5;
			var z = 123e-5;
			document.getElementById("demo2").innerHTML =
			x1 + x2 + y + z; //add line breaks

Declare (create) an Array


			var x = ["Saab", "Volvo", "BMW"];

			document.getElementById("demo3").innerHTML = x[1];

Declare (create) an Object


			var person = {
				firstName : "John",
				lastName : "Doe",
				eyeColor : "blue",
				age : 49
			};

			document.getElementById("demo4").innerHTML = person.eyeColor;

Find the Type of a Variable


			document.getElementById("demo5").innerHTML =
			typeof ""
			typeof "Joey"
			typeof "John Doe"
			typeof 0
			typeof 314
			typeof 3.14
			typeof (3.14);

			//add line breaks

Adding Two Numbers and a String

Numbers first are added and then string is appended:


			var x = 5 + 10 + "Whatever";
			document.getElementById("demo6").innerHTML = x;

Adding a String and Two Numbers

Numbers after string are treated as strings:


			var x = "Whatever" + 5 + 10;
			document.getElementById("demo7").innerHTML = x;

Undefined Variable

Both the data type and the value of a variable with no value is undefined:


			var newPerson;
			document.getElementById("demo8").innerHTML =
			newPerson + typeof newPerson;

			//add line break

An Empty Variable


			var car = "";
			document.getElementById("demo9").innerHTML =
			"The value is: " + car +  "The typeof is: " + typeof car;

			//add line break
