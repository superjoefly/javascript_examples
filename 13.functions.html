Functions

A JavaScript function is a block of code designed to perform a particular task.

A JavaScript function is executed when "something" invokes (calls) it.

A Simple Function


			function myFunction() {
				document.getElementById("demo1").innerHTML =
				"Hello World!";
			}

A Function With an Argument


			function myFunction2(name, job) {
				document.getElementById("demo2").innerHTML =
				"Welcome " + name + ", the " + job + ".";
			}

A Function With an Argument 2

When you click on one of the buttons, a function will be called. The function will display the argument that is passed to it.


			function myFunction3(txt) {
				document.getElementById("demo3").innerHTML = txt;
			}

A Function That Returns a Value


			function myFunction4(a,b) {
				return a * b;
			}

			document.getElementById("demo4").innerHTML =
			myFunction4(5,3);

Convert Fahrenheit to Celsius


			function toCel(f) {
				return (f-32) * (5/9);
			}

			document.getElementById("demo7").innerHTML =
			toCel(100);

Convert Fahrenheit to Celsius (onclick)


			function toCelsius() {

				var far = document.getElementById("input").value;
				var cel;

				if(far != '') {
					cel = (far - 32) * 5/9;
					document.getElementById("demo5").innerHTML = cel;
				}
				else {
					document.getElementById("demo5").innerHTML = "Please Enter a Number!";
				}
			}

A Function Call Without ()

Accessing a function without (), will return the function definition:


			function toCelsius(f) {
				return (5/9) * (f-32);
			}

			document.getElementById("demo6").innerHTML = toCelsius;
