Where to Insert JavaScript

In HTML, JavaScript code must be inserted between script-tags.

JavaScript can be inserted in many places:

Example: in the head section

Change paragraph text!


			function myFunction() {
				document.getElementById("demo1").innerHTML = "This paragraph was changed using JavaScript placed in the head section of the document!";
			}

Example: in the body

Change paragraph text!


			function myFunction2() {
				document.getElementById("demo2").innerHTML="This paragraph was changed using JavaScript placed in the body section of the document!";
			}

Example: external JS file

Change paragraph text!


			function myFunction3() {
			  document.getElementById("demo3").innerHTML = "This paragraph was changed using JavaScript located in an external JS file!"
			}

Example: external URL

To use a js source located at an external url, use script tags with the src = (equal to) the url. It's best to place it in the head section or at the bottom of the body section, right before the closing body-tag.


			/script src="https://www.w3schools.com/js/myScript.js">/script

Example: external folder

To use a js file that is located in a folder, include the folder name in the src. It is best to include this link in the head section or at the bottom of the body section, right before the closing body tag.


			/script src="/js/myscript.js">/script
