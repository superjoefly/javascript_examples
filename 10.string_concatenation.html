String Concatenation

When used on strings, the + operator is called the concatenation operator.

Adding Strings Together (concatenation)


			var txt1 = "Hello there...";
			var txt2 = "How are you today?";
			document.getElementById("demo1").innerHTML = txt1 + txt2;

Adding strings with space in first string


			var txt1 = "Hey there, Joe, ";
			var txt2 = "whatcha been up to?";
			document.getElementById("demo2").innerHTML = txt1 + txt2;

Adding two string together with a space in-between


			var txt1 = "Oh, my gosh";
			var txt2 = "look at her butt!";
			document.getElementById("demo3").innerHTML =
			txt1 + " " + txt2;

Adding two strings together using the += operator


			var txt1 = "Today is a good day";
			var txt = txt1 += " and I'm going to enjoy it!";
			document.getElementById("demo4").innerHTML = txt;

Adding a number and a string together returns a string.


			var x = 5 + 5;
			var y = "5" + 5;
			var z = "Hello" + 5;

			document.getElementById("demo5").innerHTML =
			x + y + z; //add line breaks
