JavaScript Dates

The Date object lets you work with dates (years, months, days, hours, minutes, seconds, and milliseconds).

Date()

Use Date() to display today's date and time:


			document.getElementById("demo1").innerHTML = Date();

getFullYear()

Use getFullYear() to display the year:


			var d = new Date();
			document.getElementById("demo2").innerHTML =
			d.getFullYear();

getTime()

Use getTime() to calculate the number of milliseconds since 1970:


			var d = new Date();
			document.getElementById("demo3").innerHTML =
			d.getTime();

setFullYear()

Use setFullYear() to set a specific date:


			var d = new Date();
			d.setFullYear(2020, 10, 10);
			document.getElementById("demo4").innerHTML = d;

toUTCString()

Use toUTCString() to convert today's date (according to UTC) to a string:


			var d = new Date();
			document.getElementById("demo5").innerHTML = d.toUTCString();

getDay() - weekday as number

Use getDay() to return the weekday as a number:


			var d = new Date();
			document.getElementById("demo6").innerHTML =
			d.getDay();

getDay() - weekday as name

Use getDay() with an array to return weekday as a name:


			var d = new Date();
			var days = [
				"Sunday",
				"Monday",
				"Tuesday",
				"Wednesday",
				"Thursday",
				"Friday",
				"Satuday"
			];
			document.getElementById("demo7").innerHTML = days[d.getDay()];
