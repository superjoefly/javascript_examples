JavaScript Variables

JavaScript variables are containers for storing data values.

Variables

In this example x, y, and z are variables:


			var x = 3;
			var y = 2;
			var z = x * y;

			document.getElementById("demo1").innerHTML = z;

Algebraic Variables

Variables can be used to compute values, just like algebra:


			var price1 = 8;
			var price2 = 9;
			var total = price1 + price2;

			document.getElementById("demo2").innerHTML = "The total is " + total;

Strings and Numbers

Strings are written with quotes; numbers are written without quotes:


			var pi = 3.14;
			var person = "Joe Atwood";
			var answer = 'Well, hello there!';

			document.getElementById("demo3").innerHTML = pi;
			//document.getElementById("demo3").innerHTML = person;
			//document.getElementById("demo3").innerHTML = answer;

JavaScript var Keyword

The var keyword is used to create variables:


			var carName; // declare a variable
			carName = "Volvo"; // assign variable a value

			document.getElementById("demo4").innerHTML = carName;   //display value of variable

Multiple Variables

Multiple variables can be declared in one statement and are separated by commas:

Note: when declaring multiple variables in one statement, you only use the var keyword once.


			var person = "Joey", carName = "BMW", price = 5000;

			document.getElementById("demo5").innerHTML =
			person + " has a " + carName + " that cost him $" + price;

Multiple Variable

Multiple variables can also be declared on multiple lines:

Note: when declaring multiple variables in one statement, you only use the var keyword once.


			var person = "Joey",
			carName = "Audi",
			price = 5000;

			document.getElementById("demo6").innerHTML =
			person + " has a " + carName + " that cost him $" + price;

Undefined

A variable without a value returns the value undefined:


			var firstCar;
			document.getElementById("demo7").innerHTML = firstCar;

Re-declaring Variables

Re-declaring a variable will NOT destroy its value:


			var newCar = "Lexus";
			var newCar;
			document.getElementById("demo8").innerHTML = newCar;

Adding Numbers

Numbers can be added together and stored in a variable:


			var x = 5 + 5 + 3;
			document.getElementById("demo9").innerHTML = x;

Adding Strings

Strings can be added together and stored in a variable:


			var name = "Joey" + " " + "Atwood";
			document.getElementById("demo10").innerHTML = name;

Concatenation (numbers as strings)

Numbers written as strings and added together will be concatentated:


			var x = "5";
			var y = "6";
			document.getElementById("demo11").innerHTML = x + y;
