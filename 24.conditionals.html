JavaScript Conditionals

Conditional statements are used to perform different actions based on different conditions.

JavaScript offers the following conditional statements:

  • if: specifies a block of code to be executed if the condition is true
  • else: specifies a block of code to be executed if the same condition is false
  • else if: specifies a block of code to be executed if the first condition is false
  • switch: specifies many alternative blocks of code to be executed
