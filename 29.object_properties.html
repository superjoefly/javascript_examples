JavaScript Object Properties

Object properties are the values associated with a JavaScript object.

A JavaScript object is a collection of unordered properties.

Properties can usually be changed, added, and deleted, but some are read only.

Accessing Properties

Access a property using .property:


			var person = {
				firstName: "Joe",
				lastname: "Blow",
				age: 37,
				eyeColor: "blue"
			};

			document.getElementById("demo1").innerHTML =
			person.eyeColor;

Access a property using [property]:


			var person = {
				firstName: "Joe",
				lastname: "Blow",
				age: 37,
				eyeColor: "blue"
			};

			document.getElementById("demo2").innerHTML =
			person["firstName"];

for...in

Access a property using for...in:


			var txt = "";
			var x;
			var person = {
				firstName: "Joe",
				lastname: "Blow",
				age: 37,
				eyeColor: "blue"
			};

			for (x in person) {
				txt += person[x] + " ";
			}

			document.getElementById("demo3").innerHTML = txt;

Adding Properties

Add new properies to an existing object:


			var txt = "";
			var x;
			var person = {
				firstName: "Joe",
				lastname: "Blow",
				age: 37,
				eyeColor: "blue"
			};

			person.car = "Toyota";
			person.weight = 233;

			for (x in person) {
				txt += person[x] + " ";
			}

			document.getElementById("demo4").innerHTML = txt;

Deleting Properties

Delete properties from an existing object:


			var txt = "";
			var x;
			var person = {
				firstName: "Joe",
				lastname: "Blow",
				age: 37,
				eyeColor: "blue",
				car: "Toyota",
				weight: 233
			};

			delete person.car;
			delete person.weight;

			for (x in person) {
				txt += person[x] + " ";
			}

			document.getElementById("demo5").innerHTML = txt;
