JavaScript Object Methods

JavaScript methods are actions that can be performed on objects.

A JavaScript method is a property containing a function definition:

Accessing a Method

Access an object method using .method:


			var person = {
				firstName: "Joe",
				lastName: "Blow",
				age: 37,
				id: 3333,
				fullName: function() {
					return this.firstName + " " + this.lastName;
				}
			};

			document.getElementById("demo1").innerHTML = person.fullName();

			var person = {

Built-In Method

Use the toUpperCase() method to convert the text to uppercase:


			var message = "Hello There!";
			var x = message.toUpperCase();

			document.getElementById("demo3").innerHTML = x;

Adding New Methods

Add a method to an object inside a constructor function:


			function person(firstName, lastName, age, eyeColor) {
				this.firstName = firstName;
				this.lastName = lastName;
				this.age = age;
				this.eyeColor = eyeColor;
				this.changeName = function(name) {
					this.lastName = name;
				}
			}
			var myFriend = new person("Robert", "Taylor", 42, "blue");
			myFriend.changeName("Bob");
			document.getElementById("demo4").innerHTML = myFriend.lastName;
