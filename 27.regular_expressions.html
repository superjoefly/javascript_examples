JavaScript Regular Expressions

A regular expression is a sequence of characters that forms a search pattern.

The search pattern can be used for text search and text replace operations.

Search for Expression

Search for an expression in a string:

Search this paragraph for the word bang!


			function myFunction1() {
				var str = document.getElementById("p1").innerHTML;
				var n = str.search(/bang/i);

				document.getElementById("demo1").innerHTML = n;
			}

Search and Replace

Search string for expression and replace it:

Search this paragraph and replace the word "bang"! (with bizang)


			function myFunction2() {
				var str = document.getElementById("p2").innerHTML;
				var txt = str.replace(/bang/i, "bizang");

				document.getElementById("p2").innerHTML = txt;
			}
