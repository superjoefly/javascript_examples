JavaScript Maths

A JavaScript Math object allows you to perform mathematical tasks on numbers.

Math.PI

Math.PI returns the ratio of a circles circumference to its diameter:


		document.getElementById("demo1").innerHTML = Math.PI;

Math.round(x)

Math.round(x) returns the value of x rounded down to its nearest integer:


		document.getElementById("demo2").innerHTML =
		Math.round(4.4);

Math.pow(x,y)

Math.pow(x,y) returns the value of x to the power of y:


		document.getElementById("demo3").innerHTML =
		Math.pow(8,2);

Math.sqrt(x)

Math.sqrt(x) returns the square root of x:


		document.getElementById("demo4").innerHTML =
		Math.sqrt(49);

Math.abs(x)

Math.abs(x) returns the absolute (positive) value of x:


		document.getElementById("demo5").innerHTML =
		Math.abs(-2.2);

Math.ceil(x)

Math.ceil(x) returns the value of x rounded up:


		document.getElementById("demo6").innerHTML =
		Math.ceil(3.3);

Math.floor(x)

Math.floor(x) returns the value of x rounded down:


		document.getElementById("demo7").innerHTML =
		Math.floor(8.7);

Math.sin(x)

Math.sin(x) returns the sin of the angle x (given in radians):

Angle in radians = (angle in degrees) * PI / 180


		document.getElementById("demo8").innerHTML = "The sin value of 90 degrees is " + Math.sin(90 * Math.PI/180);

Math.cos(x)

Math.cos(x) returns the cosin of x (given in radians):

Angle in radians = (angle in degrees) * PI / 180


		document.getElementById("demo9").innerHTML =
		"The cosine value of 0 degrees is " + Math.cos(0 * Math.PI / 180);

Math.max()

Math.max() returns the highest value in a list of arguments:


		document.getElementById("demo10").innerHTML =
		Math.max(0, 80, 300, 20, -5);

Math.min()

Math.min() returns the lowest value in a list of arguments:


		document.getElementById("demo11").innerHTML =
		Math.min(0, 80, 300, 20, -5);
