JavaScript Arithmetic

Arithmetic operators are used to perform arithmetic on numbers.

Addition (+)

Add numbers (variables) together:


			var x = 5;
			var y = 9;
			var z = x + y;

			document.getElementById("demo1").innerHTML = z;

Subtraction (-)

Subtract numbers (variables) from each other:


			var x = 32, y = 10;
			document.getElementById("demo2").innerHTML = x - y;

Multiplication (*)

Multiply numbers:


			var x = 5, y = 10;
			document.getElementById("demo3").innerHTML = x * y;

Division (/)

Divide numbers:


			var x = 10, y = 5;
			document.getElementById("demo4").innerHTML = x / y;

Modulus (remainder) (%)

Get the remainder of division:


			var x = 25, y = 10;
			document.getElementById("demo5").innerHTML = x % y;

Increment (++)

Increment a number:

x incremented and then assigned to y (y = ++x)...result of y is 11


			var x = 10;
			var y = ++x;
			document.getElementById("demo6").innerHTML = y;

x assigned to y, and then incrementd (y = x++)..result of y is 10

x assigned to y, and then x incremented (y = x++)...result of x is 10


			var x = 10;
			var y = x++;
			document.getElementById("demo7").innerHTML = y;
			document.getElementById("demo8").innerHTML = x;

Decrement

In this example, y is decremented before it is assigned to x (pre-decremented):


			var y = 5;
			var x = --y;
			document.getElementById("demo9").innerHTML = x;

In this example, y is decremented after it is assigned to x (post-decremented):

The result of y becomes 4:


			var y = 5;
			var x = y--;

			document.getElementById("demo10").innerHTML = x;
			document.getElementById("demo11").innerHTML = y;
