JavaScript Syntax

JavaScrip syntax is the set of rules defining how JavaScript programs are constructed.

Statements

Statements are separated by semicolons.

The variables x, y, and z, are assigned the values 5, 6, and 11.

Then the value of z is displayed in the paragraph below:


			var x, y, z;
			x = 5;
			y = 6;
			z = x + y;
			document.getElementById("demo1").innerHTML = z;

Numbers

Numbers can be written with or without decimals:


			document.getElementById("demo2").innerHTML = 5.5;

Strings

Strings can be written with single or double quotes:


			document.getElementById("demo3").innerHTML = "Hello JavaScript!";

Variables

In this example, x is defined as a variable. Then x is assigned the value of 6:


			var x;
			x = 6;
			document.getElementById("demo4").innerHTML = x;

Operators

JavaScript uses arithmetic operators to compute values (just like algebra):


			document.getElementById("demo5").innerHTML = (5 + 5) * 10;

Assignment

In JavaScript, the = operator is used to assign values to variables:


			var x, y;
			x = 5;
			y = 13;
			document.getElementById("demo6").innerHTML = x + y;

Expressions (using constants)

JavaScript expressions compute to values:


			document.getElementById("demo7").innerHTML = 8 * 8;

Expressions (using strings)

Expressions compute to values:


			document.getElementById("demo8").innerHTML = "Why" + " Me!";

Expressions (using variables)

Expressions compute to values:


			var x, y;
			x = 5;
			y = 8;
			document.getElementById("demo9").innerHTML = x + y;

Keywords

JavaScript keywords play a special role. The var keyword, for example, creates variables:


			var x, y;
			x = 8 + 8;
			y = x * 10;
			document.getElementById("demo10").innerHTML = y;

Comments

JavaScript comments are not executed:


			var x;
			x = 9;
			// x = 10; This comment will not be executed

			document.getElementById("demo11").innerHTML = x;

Case Sensitive

JavaScript is case sensitive...lastname and lastName are not the same:


			var lastname = "Atwood";
			var lastName = "Doe";
			document.getElementById("demo12").innerHTML = lastname;
			document.getElementById("demo13").innerHTML = lastName;
