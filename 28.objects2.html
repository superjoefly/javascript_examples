JavaScript Objects

In JavaScript, almost "everything" is an object:

  • Booleans can be objects if defined with the new keyword.
  • Numbers can be objects if defined with the new keyword.
  • Strings can be objects if defined with the new keyword.
  • Dates are always objects.
  • Maths are always objects.
  • Regular expressions are always objects.
  • Arrays are always objects.
  • Functions are always objects.
  • Objects are always objects.

All Javascript values, except primitives, are objects.

JavaScript Primitives

A primitive value is a value that has no properties or methods.

A primitive data type is data that has a primitive value.

The 5 types of primitive values are:

  • string
  • number
  • boolean
  • null
  • undefined

Primitive values are immutable (they are hardcoded and therefore cannot be changed).

Variables

Create a JavaScript variable:


			var person = "Joe Blow";
			document.getElementById("demo1").innerHTML = person;

Create Object

Create a JavaScript object:


			var person = {
				firstName: "Joe",
				lastName: "Blow",
				age: 37,
				eyeColor: "blue"
			};

			document.getElementById("demo2").innerHTML = person.eyeColor;

Single Line Object

Create a JavaScript object on a single line:


			var person = {firstName: "Joe", lastName: "Blow", age: 37, eyeColor: "blue"};

			document.getElementById("demo3").innerHTML = person.firstName + " " + person.lastName;

Multi Line Object

Create a JavaScript object on multiple lines:


			var car = {
				make: "Toyota",
				model: "Tacoma",
				color: "white",
				year: 1999
			};

			document.getElementById("demo4").innerHTML =
			"Joey has a " + car.color + " " + car.make;

Object Using New

Create a JavaScript object using new:


			var car = new Object();
			car.make = "Toyota";
			car.model = "Tacoma";
			car.color = "white";
			car.year = 1999;

			document.getElementById("demo5").innerHTML =
			"The truck is a " + car.year + " " + car.model;

Object Using Constructor

Create a JavaScript object using a constructor:


			function person(first, last, age, eye) {
				this.firstName = first;
				this.lastName = last;
				this.age = age;
				this.eyeColor = eye;
			}

			var myFriend = new person("Bill", "Davis", 57, "brown");

			document.getElementById("demo6").innerHTML =
			"My friend " + myFriend.firstName + " is " + myFriend.age + " years old.";

Built In Objects

Create built in JavaScript objects:


			var x1 = new Object();
			var x2 = new Function();
			var x3 = new Date();

			document.getElementById("demo7").innerHTML =
			"x1:" + typeof x1 + "
" +
			"x2:" + typeof x2 + "
" +
			"x3:" + typeof x3;

			//add line breaks!

There is no need to use new on String(), Number(), Boolean(), Array(), and RegExp().

Creating JavaScript Variables

It is best to create JavaScript variables together at the beginning of the script:


			var x = 5, y = 7, z = (x + y);
			document.getElementById("demo8").innerHTML = z;

Mutability

JavaScript objects are mutable...any changes to a copy of an object will also affect the original:


			var car = {
				make: "Toyota",
				model: "Tacoma",
				year: 1999,
				color: "white"
			};

			document.getElementById("demo9").innerHTML =
			car.year;

			car.year = 2001;

			document.getElementById("demo10").innerHTML =
			car.year;
