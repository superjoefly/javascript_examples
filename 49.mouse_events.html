onmousemove/onmouseout (display coordinates)

Display the coordinates of the mouse in a div while moving the mouse, and clear the cooridinates when the mouse leaves the div:


			function myFunction(e) {
				x = e.clientX;
				y = e.clientY;
				coor = "Coordinates: (" + x + "," + y + ")";

				document.getElementById("demo3").innerHTML = coor;
			}

			function clearCoor() {
				document.getElementById("demo3").innerHTML = "";
			}

onmouseover/onmouseout (change size)

Change the size of an image when the mouse moves over it, and change back to original size when the mouse moves off (out) of it:


			function bigImg(x) {
				x.style.height="150px";
				x.style.width="100px";
			}

			function normalImg(x) {
				x.style.height="100px";
				x.style.width="75px";
			}
