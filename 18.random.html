Proper Random Function

Return a random number between min (included) and max (excluded):


			/button onclick="document.getElementById('demo7').innerHTML = getRndInteger(0,10)">Click Me!/button
		

			function getRndInteger(min, max) {
				return Math.floor(Math.random() * (max - min)) + min;
			}

Return a random number between min and max (both included):


			button onclick="document.getElementById('demo8').innerHTML = getRndInt(1,10)">Click Me!/button
		

			function getRndInt(min,max) {
				return Math.floor(Math.random() * (max - min + 1)) + min;
			}
