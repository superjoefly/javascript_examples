JavaScript Examples Index

This is a collection of 60 PAGES of JavaScript EXAMPLES inspired by W3Schools. This started as a learning project, just to practice JavaScript, but I decided to make it more presentable and available to others. Each page has several examples related to the specified subject, and each example has a code snippet. I used Prism.js to style the code. Enjoy! :-)
