Popups

JavaScript has three kinds of popup boxes: alert box, confirm box, and prompt box.

Alert Box

Display an alert box:


			function myFunction1() {
				alert("Hello There!")
			}

Alert Box with Line Breaks

Display an alert box with line breaks:


			function myFunction2() {
				alert("Hello There!\nHow are you?");
			}

Confirm Box

Display a confirm box:


			function myFunction3() {
				var x;
				if (confirm("Press a button!") == true) {
					x = "You pressed OK!";
				} else {
					x = "You pressed cancel!";
				}
				document.getElementById("demo3").innerHTML = x;
			}

Prompt Box/ Alert

Display a Prompt Box and then a Response Alert


			function myFunction4() {
				var person = prompt("What is your name?");
				if (person != null) {
				alert ("Hey, there " + person + " It's nice to meet you!");
				}
			}
